# DailyBloom ✨

An motivational quote application built with Flutter that allows users to explore inspirational quotes, save their favourites, listen to narrated quotes, and improve their knowledge through an interactive quote based quiz game.


## 📱 Project Overview

QuoteVerse is a cross platform mobile application designed to provide users with daily inspiration through meaningful quotes from famous authors and thinkers. Which  allows users to browse categorized quotes, view detailed quote information, save memorable quotes, listen to quote narration, and participate in **Quote Quest**, a quiz feature where users can test their knowledge by guessing authors and completing famous quotes.


## ✨ Features

### 📝 Quote Browsing
- Display a collection of inspirational quotes.
- View quotes with:
  - Quote text
  - Author name
  - Category

### 📖 Quote Details
Users can open a detailed quote page containing:
- Full quote information
- Author details
- Quote category
- Audio narration support using local audio files
- Save and share options

### ❤️ Favourite Quotes
- Users can save quotes they like.
- Saved quotes are displayed in a dedicated Saved page.
- Users can remove quotes from favourites.

### 🎮 Quote Quest Game

An interactive quiz system where users can:

#### Guess Author
- Identify the correct author of a quote.

#### Complete The Quote
- Fill in missing words from famous quotes.

Game features:
- Multiple-choice questions
- XP reward system
- Difficulty levels
- Streak tracking

## 📦 Package Usage
-flutter_native_splash: Splash Screen
-just_audio: Audio playback and controls
-flutter_launcher_icons: Launcher Icons 

## 📱 Getting Started
1. Locate the APK at:
- build/app/outputs/flutter-apk/app-release.apk
2. Transfer the APK file to your Android device.
3. Open the APK file.
4. Enable "Install from Unknown Sources" if prompted.
5.Tap Install.
6. Launch Daily Bloom and Enjoy!
